# capacitor-swipe-back

Capacitor swipe back (ios) plugin. For Capacitor v3+. For v2 use original plugin: https://github.com/diiiary/capacitor-plugin-ios-swipe-back

## Install

```bash
npm install capacitor-swipe-back-plugin
npx cap sync
```
